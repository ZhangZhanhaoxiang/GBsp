# 国标双拼扩展到三拼

### 介绍
依据GB/T 34947-2017标准；包含GB2313所有汉字；符合手心输入法双拼辅助码格式.
|a|e|s|o|l|
|--|--|--|--|--|
|阴平|阳平|上声|去声|轻声|

### 使用教程

以手心输入法安卓端v3.5.3.1208为例：
1.  将“国标双拼辅助码.dat”文件下载到手机存储目录；
2.  打开手心输入法APP；
3.  点击“我”->“设置”->“输入设置”->“双拼”->“辅助码”；
4.  默认辅助码是自然码的辅助码，这里选“自定义辅助码”；
5.  点击刚刚下载的辅助码文件，提示“双拼辅助码导入成功”.
